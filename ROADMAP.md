# Roadmap: Адаптация Telegram Web App под Desktop

Цель: Сделать так, чтобы окно веб-приложения на компьютере (Telegram Desktop) автоматически "отдавалось" (масштабировалось) на 90% (или другое значение), чтобы весь контент влезал без скролла, сохраняя идеальный вид на мобильных устройствах.

## Текущая проблема
На мобильных устройствах верстка отображается корректно. На ПК (Desktop) стандартное окно Telegram слишком узкое/низкое для текущего размера шрифтов и отступов, из-за чего пользователю приходится вручную делать масштаб 90% (Ctrl + колесико).

## План реализации (Step-by-Step)

### Шаг 1: Определение платформы (JavaScript)
Необходимо определить, с какого устройства зашел пользователь. Telegram WebApp предоставляет параметр `platform`.

**Где внедрять:** В `index.html` внутри тега `<script>`, где инициализируется `tg`.

**Код для внедрения:**
```javascript
const platform = tg.platform; // 'android', 'ios', 'tdesktop', 'macos', 'web'
// Добавляем класс к body, если это десктоп
if (['tdesktop', 'desktop', 'macos', 'web', 'weba', 'unknown'].includes(platform)) {
    document.body.classList.add('platform-desktop');
}
```

### Шаг 2: Применение масштабирования (CSS)
Используем CSS-свойство `zoom` или `transform: scale()`, чтобы уменьшить интерфейс только для десктопа. Свойство `zoom` работает в Chrome/Webkit (на чем работает Telegram Desktop) идеально и влияет на лейаут, "сжимая" его.

**Где внедрять:** В секцию `<style>` в `index.html`.

**Код для внедрения:**
```css
/* Только для десктоп версии */
body.platform-desktop {
    /* Вариант 1 (рекомендуемый для Telegram Desktop): Используем zoom */
    zoom: 90%; 
    
    /* Вариант 2 (если zoom работает некорректно): Transform */
    /* 
    transform: scale(0.9);
    transform-origin: top center;
    width: 111.11%; Calculate: 100% / 0.9 = 111.11%
    height: 111.11%;
    */
}

/* Опционально: Дополнительные правки для десктопа чтобы убрать лишние отступы если зума мало */
body.platform-desktop .bg-white {
    /* Например, чуть уменьшить скругления или тени если нужно */
}
```

### Шаг 3: Тестирование
1. **Мобильное устройство:** Открыть бот. Проверить, что класс `platform-desktop` **НЕ** добавился (через отладку или визуально - масштаб не должен измениться).
2. **ПК (Telegram Desktop):** Открыть бот. Проверить, что масштаб стал меньше (как при ручном 90%). Проверить, что весь контент влезает (кнопка "Далее" видна без скролла).

## Итог
Данное решение не затрагивает мобильную версию (так как класс добавляется только на десктопе) и автоматически решает проблему масштаба на ПК.
