# Инструкция по откату изменений (Rollback)

Если после обновления что-то пошло не так, выполните следующие действия:

## 1. Локальный откат
Чтобы вернуть старый `index.html`:
1. Откройте терминал.
2. Введите команду:
   ```bash
   copy webapp\index.html.bak webapp\index.html
   ```
   (Подтвердите замену, если спросит: `Yes`)

## 2. Откат на сервере (Railway)
После локального отката нужно отправить изменения на сервер:
1. Введите команды:
   ```bash
   git add webapp/index.html
   git commit -m "revert: layout changes"
   git push origin main
   ```
2. Дождитесь деплоя (минута-две).

Все вернется как было.
